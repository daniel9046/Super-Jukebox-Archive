Title Bar

The Title Bar at the top of the main window is divided into three parts. If a song is playing or is paused then the first information displayed is the name of the song, if the song contains a title in its ID666 tag then that title will be used, otherwise the file name of the song will be used. The next part contains the name of the currently open playlist, an asterisk will be tagged onto the playlist name if it needs to be saved. And the last part contains the name of the program.

The Title Bar can be enabled/disabled from either the View menu or the View sub-dialog in the main program Configuration dialog box or by pressing Ctrl+1.
