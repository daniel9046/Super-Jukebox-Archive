Play List

The Play List displays all the SPC files that are currently inside the playlist. New entries that are added are automatically sorted by their names and entered to the bottom of the playlist. You can click on a column header to resort them in a specific order, either ascending or descending. If a sort is in effect new entries that are added are automatically sorted before they are displayd. Also note that a playlist filter could be in effect, in which case the entries being added are automatically filtered before being displayed. Individual columns can be rearranged in different locations by draging a column header horizontally. Individual columns can also be enabled and disabled through the main program Configuration dialog box via the View sub-dialog. Right-click or press the Context Menu key to pop-up the context menu for the playlist entries. If you click an entry that is already selected it will activate the edit box for that entry. If you are currently editing an entry there are several ways to navigate to other entries for editing, simply click on another entry and the edit box will move to that entry, press the Tab key to move to the next sub-entry to the right, hold down the Ctrl key while pressing Tab to move to the next entry to the bottom, hold down the Shift key while pressing Tab or Ctrl+Tab to move to the previous sub-entry or entry, respectively.

The Play List can be enabled/disabled from either the View menu or the View sub-dialog in the main program Configuration dialog box or by pressing Ctrl+5.
