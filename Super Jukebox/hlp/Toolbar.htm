Toolbar

The Standard Buttons Toolbar contains push buttons that can be pressed using the mouse to perform designated actions. When the mouse pointer hovers over a button a tooltip window will apear giving a descrition for the button, it is also displayed in the fourth pane of the main window status bar. This toolbar can display buttons either with or without pictures and with or without text labels or both, it can also be specified to have its buttons wrap on multiple rows when there is not enough width to fit them all and you do not wish to scroll through them. The Standard Buttons Toolbar can also be customized as to what buttons and in which order you want to have displayed by right-cliking on it and selecting the Customize... command.

The Standard Buttons Toolbar can be enabled/disabled from either the View menu or the View sub-dialog in the main program Configuration dialog box or by pressing Ctrl+2.
