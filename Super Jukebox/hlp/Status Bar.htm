Status Bar

The Status Bar is partitioned into five panes. The first pane displays the number of the song that is currently playing and the number of total songs displayed in the Play List. The second pane displays the elapsed time for the currently playing song and the length if one is in effect. The third pane is a graphical progress bar that represents the elapsed time and is only drawn if a song length is in effect, if you move the mouse over this progress bar a seek-to time is displayed over it in relation to the position of the mouse cursor if you then click on it the song will seek to that time. The fourth pane contains auxilary information such as menu item descriptions and the current activity of the program. The fifth pane indicates whether or not a playlist filter is in effect. The sixth pane displays the current Preamp level which can change dynamicaly depending on the Automatic Preamplification Reduction setting.

The Status Bar can be enabled/disabled from either the View menu or the View sub-dialog in the main program Configuration dialog box or by pressing Ctrl+8.
