User Interface

The basic usage of Super Jukebox is very simple, usually it simply consists of finding your SPC files with the File List and adding them into the Play List, then you can play them, edit them, sort them, fitler them, shuffle them, etc...

Under this folder are entries that describe the main window components of Super Jukebox. Remember that every dialog box in Super Jukebox has context sensitive help for its controls, just click on the "?" button in the upper-right hand corner of the dialog box then click on the control you are interested in.

And please remember to have fun while using Super Jukebox!
