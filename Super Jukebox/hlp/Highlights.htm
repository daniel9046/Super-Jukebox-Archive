Highlights

Installation
  • Installation and uninstallation programs.
  • Total of three SPC700 engines, Uematsu (in Super Jukebox), Snes9X and ZSNES (in MCI driver).
  • DLL (Uematsu.DLL) for any and all programs to use. Header and import library provided.
  • MCI driver (MCISPC.DRV) for any and all programs to use (Media Player).
  • Can switch between the Snes9X and ZSNES SPC700 engines in the MCI driver.
File Support
  • All SP* files.
  • Can extract SPC files from ZIP archives. ZIP offers fast decompression.
  • Can extract SPC files from RAR archives. RAR offers *great* commpression for SPCs.
  • Can extract SPC files from ACE archives.
  • Uses playlists.
  • Export playlist to HTML files.
  • Full support for ID666 tags.
  • Export playlist entries to SPCs' ID666 tags.
  • Drag-and-drop folders and files support.
Playlist Handling
  • Playlist can be sorted by any ID666 tag.
  • Playlist columns can be moved.
  • Normal search feature.
  • Playlist filter feature with powerfull commands and history lists.
  • New shuffle feature.
  • Auto resize columns.
  • Background picture (with customizable blending level).
Player
  • Flexible support for the latest and future versions of SNESAPU.DLL.
  • Detects inactivity if the song stopped playing.
  • Supports up to 32-bit/96kHz/Surround sound.
  • Optimized for x86, MMX, 3DNow! and SSE processors.
  • Allows saving as WAV file.
User Interface
  • Fully customizable, extensively configurable.
  • Usefull toolbar customize screen with own icons for nearly all needs.
  • Location bar with history list for most recently visited folders.
  • On-the-fly toggling of playlist, filelist, visualizer and nearly everything else.
  • Full Help support.
  • Super fast, tight and accurate visualization.
  • Support for additional visualization plugins through Active Wallpaper interface.
  • Fully working filelist with all common features.
  • Full support for keyboard shortcuts.
