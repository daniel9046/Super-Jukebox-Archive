File List

The File List displays all the available drives, directories and files supported by Super Jukebox. They are automatically sorted in ascending alphabetical order by their names. You can double-click or press Enter to perform the default action for an item. For drives, directories and archvies the default action is to switch into them. For SPC files it is to add them to the playlist. For playlist files it is to load them into the program. Right-click on an entry or press the Context Menu key to pop-up the context menu, individual columns can be enabled/disabled via this menu. To perform the alternate action on an entry hold down Shift while pressing Enter or double-clicking, for drives, directories and archives this will add them to the playlist, for SPC files this will play them.

The File List can be enabled/disabled from either the View menu or the View sub-dialog in the main program Configuration dialog box or by pressing Ctrl+4.
