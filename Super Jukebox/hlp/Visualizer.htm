Visualizer

The Visualizer with the exception of the eight check boxes is only for output. Use the eight corresponding check boxes in the Active column to enable/disable an individual channel. The rest of the columns should be self-explanatory. Right-clicking on the Visualizer produces the context menu where you can select the Configure... command to adjust the settings for it.

The Visualizer can be enabled/disabled from either the View menu or the View sub-dialog in the main program Configuration dialog box or by pressing Ctrl+7.
