Location Bar

The Location Bar is composed of a push button labeld "Up One Level" that when clicked will move you up in the directory hierarchy, this button will be disabled when at the top of the hierarchy which contains a list of the system drives. The combo box to the right of the push button displays the current location of the File List and it's corresponding icon, you may type your desired location in this combo box and it will be added at the top of its history list, the history list can hold at most ten entries and beyond that old ones will be discarded.

The Location Bar can be enabled/disabled from either the View menu or the View sub-dialog in the main program Configuration dialog box or by pressing Ctrl+3.
